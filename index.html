HTML Lesson 1: The Power of Semantic Tags

What is Semantic HTML? ü§î

**Semantic** means **"relating to meaning in language or logic."** A semantic element is a tag that clearly describes its meaning to both the **browser** and the **developer**.

Non-Semantic Tag Semantic Tag Meaning to the Browser/Developer
<div> <header> Introductory or navigational content.
<div> <main> The primary, unique content of the page.
<div> <article> A self-contained piece, like a blog post.

Code Example: Semantic (Good) Structure

<header>
    <h1>My Site</h1>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <h2>Post Title: Semantic HTML</h2>
        <p>This is the self-contained article content.</p>
    </article>
</main>

<footer>
    <p>&copy; 2025 ICT Code Lessons</p>
</footer>

CSS Lesson 1: Mastering The Box Model

The Four Components of the Box Model üì¶

Every HTML element is a rectangular box made of four layers, from the inside out: **Content, Padding, Border, and Margin.**

  • **Content:** The actual text or image (controlled by `width` and `height`).
  • **Padding:** The **internal space** between content and border (takes the element's background color).
  • **Border:** The line wrapping around the padding and content.
  • **Margin:** The **external space** between this element and other elements (always transparent).

CSS Code: Applying Box Properties

We're styling an <h1> element to demonstrate all four components. 

h1 {
    /* 1. Content Width */
    width: 80%; 

    /* 2. Padding (Internal Space) */
    padding: 20px; 

    /* 3. Border (The Line) */
    border: 5px solid darkblue;
    
    /* 4. Margin (External Space) */
    margin: 40px auto; 
    /* 40px top/bottom; 'auto' horizontally centers the block */

    /* Decoration */
    background-color: #1e40af; /* Blue-700 */
    color: white;
    text-align: center;
}

The Crucial Trick: box-sizing: border-box

The default browser setting (`content-box`) adds padding and border **outside** the defined width, breaking layouts. **Always** use this professional standard to make padding and border calculate **inward**: 

/* The 'Universal Selector' targets ALL elements */
* {
    box-sizing: border-box; 
}

The Flexbox Fundamentals: Container and Items

**Flexbox** is the modern solution for **one-dimensional layout** (arranging items in a row or column). It requires setting a parent element as the **Container** and allowing it to control the alignment of its children **Items**.

Element Key Property What it Controls
**Flex Container** (Parent) display: flex; Overall alignment and distribution of children.
**Flex Item** (Children) flex-grow, flex-basis The size and order of the individual items.

CSS Code: Setting the Container

We use a <nav> as the container for three links (items). 

.main-menu {
    display: flex; /* <-- Activates Flexbox! */

    /* 1. Alignment along the Main Axis (Horizontal) */
    justify-content: space-between; 

    /* 2. Alignment along the Cross Axis (Vertical) */
    align-items: center; 
    
    /* Decoration */
    height: 60px;
    background-color: #333;
}

Key Container Properties Explained

  • `justify-content:`: Controls **horizontal** spacing (`center`, `space-between`, `flex-end`).
  • `align-items:`: Controls **vertical** spacing (`center`, `flex-start`, `stretch`).
  • `flex-direction:`: Changes the axis (`row` or `column`).

JavaScript Lesson 1: DOM Manipulation üß†

The **DOM (Document Object Model)** is the tree-like map that the browser creates from your HTML. JavaScript doesn't see your HTML tags as text; it sees a structured, interactive map of **objects**. Every element‚Äîthe <body>, the <p>, the <h1>‚Äîis an object that JavaScript can read, change, delete, or create.

To change anything on the page, you always follow the same three steps in JavaScript: **1. Select** the element, **2. Act** on it (change its text/style), and **3. Listen** for a user action (like a button click) to trigger the change.

Code Example: Changing Text on a Button Click

Status: Waiting for interaction...

HTML Setup:

<section id="js-lesson-1-dom">
    <h3 id="status-text">Status: Waiting for interaction...</h3>
    <button id="action-button">
        Click Me to Activate JS!
    </button>
</section>

JavaScript Code:

This script runs the core logic. Note how we use document.getElementById for selection. 

// Step 1: Select the elements we need to interact with
const statusText = document.getElementById('status-text');
const actionButton = document.getElementById('action-button');

// Step 3: Listen for a user action (a 'click' event) on the button
actionButton.addEventListener('click', () => {
    
    // Step 2: Act (The function that runs when the button is clicked)
    
    // 2a. Change the text content of the h3 element
    statusText.textContent = 'Status: JavaScript Activated!';
    
    // 2b. Change the style (CSS) of the h3 element
    statusText.style.color = '#10B981'; // Tailwind's emerald-400 color
    
    // 2c. Add a new CSS class (Tailwind class for emphasis)
    statusText.classList.add('underline');
    
    // Prevent the function from running again
    actionButton.disabled = true;
    actionButton.textContent = 'Activated!';
    actionButton.classList.remove('bg-indigo-600', 'hover:bg-indigo-700');
    actionButton.classList.add('bg-gray-500', 'cursor-not-allowed');
});
